# TESTE SAE DIGITAL em PHP - Bootstrap, PDO & MySQL

Desenvolvimento de uma aplicação que gerencie os espetáculos e as reservas das poltronas de um teatro. 

- Linguagem PHP.
- Acesso a banco de dados com o MySql
- Otimização da conexão com o banco de dados através do PDO (PHP Data Object)
- Uso de linguagens, como: JavaScript e CSS
- Uso do framework Bootstrap para realização de um layout responsivo para o projeto.

## Configuração do Projeto:

- Executar a query sae.sql ou importar o arquivo no phpMyAdmin para criar a table necessária.
- Editar o arquivo **banco.php** 

```
$dbNome = 'nomeDaTable' 
$dbHost = 'nomeDoDominioOuIP:Porta' 
$dbUsuario = 'usuarioDoMysql' 
$dbSenha 'senhaDoUsuario'

```
